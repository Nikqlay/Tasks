Если взять натуральные числа, которые меньше 10 и кратны 3 или 5, то мы получим следующие значения: 3, 5, 6, 9. Сумма этих чисел равна 23.

Напишите функцию, которая должна возвращать сумму всех чисел, которые кратны 3 или 5 и меньше переданного в функцию числа.

function solution($number){
$s=0;
for ($i=1; $i <$number; $i++) {
    if ($i%3 ==0 || $i%5 ==0) {
	$s=$s+$i;
	}
	  
}
return ($s);

}
