Вам задан массив, который содержит целые числа. 
Массив полностью состоит из целых четных или 
нечетных чисел, за исключением одного числа N. 
Напишите метод, который принимаем массив, 
как аргумент и возвращает число N.

<?php
$arr = [1, 16, 1, 1];
$arrp = [];
$arro = [];
$i=0;
foreach ($arr as $value) {
    if ($value%2 ==0){
	$arrp = $value ;
	$i++;
	}
	else {
	$arro = $value;
	}
}
if ($i > 1) {
	var_dump ($arro);
}
else {
	var_dump ($arrp);
}

?>
